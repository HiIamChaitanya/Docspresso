# 📄 Docspresso
Ok so here is the rough idea: Docspresso is application for mostly for Linux may be for windows as well, I'm starting this project as i feel there exists a gap when using Linux for some non tech stuff such as dealing with documents it gets some what non-linear and is frustrating. basically can not really on online apps for my sensitive data and Linux does not have any one app that does the job, have to switch 50 different apps to do one dam thing, WTF!.
sorry, but anyway I'm try to do just that with this project which may or may not be idea similar to Adobe acrobat but for Linux.

## not key features:
- 🔒 P2P encrypted local network operations some what similar to syncthing but opens port to share file temporarily 
- 🎨 Modern, intuitive user interface

## Document Handling Capabilities:
- ✂️ Compress
- 🔗 Merge
- ✏️ Edit
- 🔄 Rearrange
- 🎭 Redact PDF
- 🖼️ Crop
- 🎨 Apply filters to Scaned pdf or scanned documents
- PDF OCR
- convert file formats
- change aspect ratio

## Supported File Formats:
- 📑 PDF
- 🖼️ PNG, JPEG, JPG
- 🎨 SVG
- TeX
- Markdown
- docs and word,pptx ,etc
...and more!

## not so advanced features:
- 🧠 Ollama integration for PDF files and 🔍 Document scanning functionality
