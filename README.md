# 📄 Docspresso
Ok so here is the rough idea: Docspresso is an application for mostly for Linux may be for windows as well, I'm starting this project as I feel there exists a gap when using Linux for some non-tech stuff such as dealing with documents it gets somewhat non-linear and is frustrating. Basically can not rely on online apps with my sensitive data or any document files, therefore need local app and Linux does not have any one app that does the job, have to switch 50 different apps to do one damn thing, WTF!.
Sorry, but anyway I'm trying to do just that with this project, which may or may not be an idea similar to Adobe Acrobat but for Linux.

## not key features:
- 🔒 P2P encrypted local network operations somewhat similar to syncthing but opens port to share file temporarily 
- 🎨 Modern, intuitive user interface

## Document Handling Capabilities:
- ✂️ Compress
- 🔗 Merge
- ✏️ Edit
- 🔄 Rearrange
- 🎭 Redact PDF
- 🖼️ Crop
- 🎨 Apply filters to Scanned PDF or scanned documents
- PDF OCR
- convert file formats
- change aspect ratio

## Supported File Formats:
- 📑 PDF
- 🖼️ PNG, JPEG, JPG
- 🎨 SVG
- TeX
- Markdown
- docs and word, pptx ,etc
...and more!

## not so advanced features:
-  Ollama integration for PDF files and 🔍 Document scanning functionality
